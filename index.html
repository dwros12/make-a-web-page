Lesson 1: The Basics of the Web and HTML

How the World Wide Web Works

The web is a bunch of computers that communicate with each other. When you open a browser and go to a web page like www.google.com or www.yahoo.com , your computer sends a HTTP Request across the internet to a server. The server finds the appropriate HTML document and sends it back to the your computer where your web browser interprets the page and displays it on the user's screen. See this video for addition explaination.

HTML

HTML stands for Hypertext Markup Language. HTML documents are the main type on the web. HTML documents contain text content which describes "what you see" and markup which describes "how it looks". This video gives a good overview.

Tags, Elements and Attributes

HTML documents are made of HTML Elements, Tags, and Attributes. Elements represents a structure and has a start tag, content, and end tag. Tags are used at the beginning and end of HTML. Attributes are used to define a property. This video explains the distinction well.

Technological Empathy

Computers interpret instructions literally. They cannot interpret the intent of the programmer. This can cause huge problems in the program and make a small mistake a huge problem.

Inline vs Block Elements

HTML elements are either inline or block. Block elements applies only to the content inside of them. While Inline applies to the entire section of code. Example Break is inline while Paragraph is block.

Lesson 2: Creating a Structured Document with HTML

Programming Laugages of Web Development

There are three programming laugages of Web Develpment. HTML, CSS, and JavaScript. HTML creates the structure of the webpage, while CSS (cascading style sheets) control the style of the webpage and JavScript provides the interactive components of the page.

Document Object Model

A DOM is how you draw out what you want the webpage to look like without having to write any code. It's a great way to visual what you need to accomplish and communicate to non-technical users. The DOM is tree-like structure of HTML is from a Parent-Child relationship. A Parent can have multiple children and the children can have multiple children, etc.

Boxes

Everything on the webpage is a box. There are boxes inside of boxes inside of boxes. The more boxes a part of the webpage is in the more nested it is in the code.

Text Editors for HTML

There are three Text Editors that are recommended for the course.
1) Scratchpad
2) CodePen
3) Subline
Scratchpad is best for beginners and is the most user friendly, while Sublime is more powerful and less user friendly, while Codepen is in the middle of them.

Lesson 3: Adding CSS Style to HTML Structure

Avoiding Repetition

Avoiding repetition allows for a number of benefits and will help you program better.

Helps you avoid errors. When you have to repeat code and you run a good chance of either missing a place you need to put it or typing the code wrong. Also if you need to make a minor change you will need to go to multiple spots to make the changes and that can be very time consuming.

By not repeating code it is easier to give all your webpages and websites the same look and feel and allow for the same consistency across the organization. If you repeat a lot of code it will be a lot harder to generate the same consistency.

CSS

CSS stands for Cascading Style Sheets and it allows you to adjust the stype of HTML elements. By specifing which HTML elements are similar and puting in the same class you can give them a specific style.
